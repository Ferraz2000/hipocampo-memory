# plugin/ — Claude Code plugin (WIP, Phase 4)

This will hold the Claude Code plugin: generator skills (`brain-init`,
`brain-router-init`, `brain-scripts-init`, `brain-update`), workflow skills
(`registra`, `discovery`, `spec`, `busca`, …) in the open Agent Skills format,
hooks (`SessionStart` git-derived briefing, `Stop` capture-sweep), and the
`.claude-plugin/plugin.json` manifest. A sibling `.claude-plugin/marketplace.json`
at the repo root will make it installable via `/plugin marketplace add`.

Empty until Phase 4 — see [`../PLAN.md`](../PLAN.md).
